## 🌟 Clash for Android 核心特性

* **规则分流**
  基于域名、IP、GeoIP 的分流规则，完美解决国内直连与海外代理的智能切换。
* **分应用代理**
  可自定义哪些 App 走代理，哪些 App 直连，灵活管理手机流量。
* **DNS 优化**
  内置 DNS 解析优化，防止 DNS 污染，提升网页加载速度。
* **实时监控**
  在通知栏或 App 内实时查看当前下载/上传速度及流量消耗。

## ⚙️ Clash for Android 配置与订阅导入教程

即使项目停止更新，其 **配置逻辑与标准 Clash 订阅完全通用**。

### 一、导入订阅链接
1. 打开 Clash for Android，点击 **配置 (Profiles)**。
2. 点击右上角的 **+** 号。
3. 选择 **从 URL 导入 (URL)**。
4. 在名称处填写备注（如机场名），URL 处粘贴你的 **Clash 订阅链接**。
5. 点击右上角磁盘图标保存，等待下载配置。
6. 返回配置列表，选中刚才添加的配置文件。

### 二、开启代理
1. 返回 App 主界面。
2. 点击中间的 **已停止 (Stopped)** 按钮。
3. 首次开启会弹出 VPN 权限请求，点击 **确定 (OK)**。
4. 按钮显示为 **运行中 (Running)** 即代表已成功开启。

## ❓ 常见问题

### 导入订阅失败怎么办？
1. 请确认订阅链接是否为 Clash 格式（非 SSR/V2Ray 链接）。
2. 尝试切换当前网络（使用手机流量或开启其他临时代理）后重新下载。
3. 检查手机系统时间是否与北京时间同步。

### 电池消耗过快？
Clash 需要实时处理流量分流，会有一定的电量消耗。建议在 **设置 -> 电池优化** 中将 Clash 设置为 "不优化"，以防止系统在后台强制杀掉进程导致断网。
